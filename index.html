Data Visualization Design at SON Company
Jan 2015 - Oct 2016

About

I started Data Visualization design as an internship in SON company when I was still a second-year student at university in order to gain work experience and satisfy requirements for qualification. In this project, we created an interactive data visualization for Pasargad Oil Company, the largest producer and exporter of bitumen in Iran. The dashboard`s aim was to define and analyze performance indicators in the form of reports and managerial dashboards in various functional areas (commerce, finance, human resources, production and operations, strategy, etc).



Objectives

  • Provide reports for users of different groups based on existing indicators.
  • Add new indicators and create corresponding report.
  • Mechanize the receiving of required information for reports, which is currently executed in manual procedures.
  • Ability to add other data sources to the system by the system administrator.

Formal Method Course
Heart Health Gadget

About

As our final project in the Formal Method course, We examined the function of heart health instruments Using a circular instrument attached to the skin we used internal sensors to monitor heart rate and blood pressure. This was transmitted through Bluetooth to external devices for complete results. To start we initiate with conceptual, activity, and class diagrams, then we implemented the general form of the problem using the Z language and converted the class diagram into Z language schemes. In the end, we simulate it with CPN Tools software to see how this device works using the defined Token. We also used GCL (Guarded Command Language) to be able to view the conditions defined in the gadget and to examine its loops. Finally, we defined the whole project as a class and simulated it with Alloy programming language, and examined the connection between classes.
